# Tax
This file is used to calculate how much you will have to pay given your taxes, **assuming** you're in the single tax bracket.

## Tax Bracket

| Tax Rate | Single Taxable Income |
| -------- | --------------------- |
| 10%      | $0-$9,275             |
| 15%      | $9,276-$37,650        |
| 25%      | $37,651 – $91,150     |
| 28%      | $91,151 – $190,150    |
| 33%      | $190,151 – $413,350   |
| 35%      | $413,351 – $415,050   |
| 39.6%    | $415,051+             |
